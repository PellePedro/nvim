# File Navigation

## Open Treeview (nvim-tree.lua)
```
Shortcuts
=========
<ctrl-n>	Opens Treeview (in normal mode)
<ctrl-l>	Move from treeview to editor view
<ctrl-h>	Move from editor view to tree view

```

## Open File (fzf)
```
Shortcuts
=========
<ctl+p>
```

## Show Commits
```
Shortcuts
=========
gc
```

## Git Status
```
Shortcuts
=========
gs
```

## Toggle between the two last files (a.k.a alternative)

```
Shortcuts
=========
<sp + a>
<ctl + ^>
```

## Quick fix window

The Quick fix window is a listview populated with references when invoking LSP References (gr)

```
Shortcuts
=========
[Q  Goto first entry in quick fix list
[q  Goto previous entry in quickfix list
]q  Goto next entry in quick fix window
]Q  Goto last entry in quick fix window
\q  Close quick fix window
```
