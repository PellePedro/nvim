# Nvim Configuration



## Install Neovim

- On Mac

  ```
  brew install neovim
  ```
- Fedora

  ```
  sudo dnf install neovim
  ```

- Ubuntu

  ```
  sudo apt install neovim
  ```


## Clone this repo into your config

```
git clone https://github.com/pellepedro/nvim.git ~/.config/nvim
```

## Install python & node support

```
pip install pynvim
```

```
npm i -g neovim
```


## List of programs you should install

- ripgrep
- fzf
- lazy git
