# Key Mappings


# Leader Mappings

```
-finding
<leader>ff 	Find Files
<leader>fw 	Grep for word under cursor
<leader>fc 	Find files under ~/.config/nvim
<leader>fs 	Grep with input
<leader>fb 	List Buffers

-explore
<leader>e 	Open Vifm

<leader>gc 	Git Commits
<leader>gbc Git Buffer Commits
<leader>gs 	Git Status

<leader>y 	Yank Visual Selection to System Clipboard OSCYank
<leader>1 	Show LSP Dockument Symbols
<leader>2 	Show LSP Workspace Symbols
```

# Normal Mode Mappings

## g Mappings

```
gd : Goto Definition
gh : Show Documentation
gf : Goto File
gr : Show references
gc : COC fix current
gy : Goto Type
gn : COC diagnostic next error
gp : COC diagnostic previous error
[g : COC diagnostic next
]g : COC diagnostic previous


```

## \ Mappings

```
\\ Show only this pane
\v Open $HOME/.config/nvim/init.vim
\d Show LSP diagnostics
\t Show Type information
\f Show Type information in fzf
\q Close type information
```


## 'space' mappings
```
<space>a : Alternative File (<c-^>)
<space>b : List Buffers in fzf
<space>c : List commits
<space>l : Show none printable characters
<space>s : Show git status
<space>u : Show Undotree
<space>w : Trim trailing whitespaces
<space>y : Yank to ocs52

```


## 'Control' mappings
```
<c-s>  : Save Buffer
<c-q>  : Quit (:q)
<c-[]> : Escape

```
